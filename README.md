# 蒋政雷 · 个人简历

> AI应用开发 · 算法竞赛 · 全栈实践

基于 Apple 极简美学设计的个人简历网页，支持在线编辑模式。

## 在线地址

https://github.com/Jiang0810/Jiangzhenglei

## 技术栈

- 纯前端 HTML + CSS + JavaScript
- CSS Custom Properties 实现浅色/深色主题自适应
- Apple SF Pro 字体体系，极简设计语言
- 无外部依赖，轻量可部署

## 功能特性

- **全屏翻页**：鼠标滚轮/触摸滑动逐屏切换
- **逐项渐入**：内容按顺序淡入动画
- **项目详情浮层**：点击卡片查看完整项目描述，支持展示链接与视频上传
- **编辑模式**：右下角 ✎ 按钮进入编辑，支持文字直接修改、图片替换，保存后下载 HTML
- **深色模式**：跟随系统 `prefers-color-scheme: dark` 自动切换

## 内容结构

1. **首页** — 姓名 + 简介
2. **关于本人** — 教育背景、竞赛获奖
3. **实习与项目** — 实习经历 + 项目作品（3×2 卡片网格）
4. **技能水平** — 专业技能清单
5. **联系方式** — 邮箱、电话、微信、所在地

## 使用说明

### 编辑内容

点击右下角 ✎ 按钮进入编辑模式：
- 直接点击文字进行修改（蓝色虚线高亮）
- 点击图片可替换为本地文件
- 点击 💾 保存并下载更新后的 HTML

### 添加项目展示链接

在 `js/global.js` 中找到对应项目的 `showcaseLink` 字段，填入 URL：

```js
showcaseLink: 'https://your-demo-link.com'
```

### 添加演示视频

方式一：将视频文件放入 `images/videos/` 目录，然后在 `js/global.js` 中设置：

```js
video: 'images/videos/demo.mp4'
```

方式二：打开项目详情浮层，点击视频占位区域直接上传。

## 本地运行

直接在浏览器打开 `index.html` 即可预览。
